# html-frame

html frame

## Simple HTTP Server 설치및 실행

```
// http-server
// live reload가 안됨. 개발시 불편함.

// 설치
npm install -g http-server

// 실행
http-server
```

## Live Server 설치및 실행

```
// http-server
// live reload가됨.

// 설치
npm install -g live-server

// 실행
live-server --port=8080 --browser=chrome
```
